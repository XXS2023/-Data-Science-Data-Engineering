# -Data-Science-Data-Engineering

Этот проект посвящен анализу зарплат специалистов в области Data Science и Data Engineering. Целью проекта является выявление тенденций в зарплатах, а также факторов, влияющих на уровень вознаграждения специалистов в этих областях.

## Технологии

Проект реализован с использованием следующих технологий:

- Python
- Pandas
- Matplotlib / Seaborn (для визуализации)
- Jupyter Notebook (для анализа)
- Statsmodels (для анализа)
- Scipy (для анализа)

## Данные

[Оригинальный датасет](https://www.kaggle.com/datasets/ruchi798/data-science-job-salaries)


## Анализ

В рамках проекта был проведен анализ данных с использованием статистических методов и визуализации. Были выявлены следующие ключевые аспекты:

- Тенденции роста зарплат с течением времени.
- Сравнение зарплат между профессиями.
- Влияние размера компании на уровень зарплат.
- Структура занятости по должностям.

## Выводы

Результаты анализа подтверждают растущий спрос на специалистов в области Data Science и Data Engineering, а также выявляют факторы, влияющие на уровень их зарплат. Более подробные выводы можно найти в соответствующем разделе анализа.

## Использование

1. Клонируйте репозиторий на свой локальный компьютер:

bash
git clone https://github.com/XXS2023/Data-Science-Data-Engineering.git

2. Перейдите в директорию проекта:

bash
cd Data-Science-Data-Engineering

3. Запустите Jupyter Notebook для анализа данных:

bash
jupyter notebook

4. Откройте файл EDA_ds_salaries.ipynb и следуйте инструкциям.
